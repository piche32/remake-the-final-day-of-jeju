# 제주도 최후의 날
Unity 2019.3.5로 만든 제주도 최후의 날을 Unity 6.3.11버전으로 리메이크

## 리메이크 시 추가/변경 기능
- 멀티플레이 모드

## 개요
2030년, 휴화산으로 알려졌던 한라산이 활동을 한다는 정황이 보고 되었다. 화산이 언제 터질지 모르는 급박한 상황 속에서 사람들 대신 안드로이드가 식량을 찾아 섬의 이곳저곳을 돌아다닌다. 한라산이 폭발하기 전에 음식을 챙겨 항구를 통해 탈출하자.

- 장르: 필드 액션
- 플랫폼: 모바일
- 플레이 인원: 1인 / 멀티

### 플레이 방식
- 조이스틱: 이동
- 액션 버튼

## 사용 프로그램
- Unity 6.3.11
- Visual Studio Code
- Gemini CLI
